ENV Variables in Docker
A) Using -e flag
docker run -e DB_HOST=localhost -e DB_PORT=5432 myapp

B) Using Dockerfile
ENV APP_ENV=production
ENV DB_HOST=postgres

C) Using --env-file
docker run --env-file .env myapp
